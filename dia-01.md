# Dia 1 - Conceitos Básicos de Redes

## O que é uma rede?

Uma rede é um conjunto de dispositivos conectados para compartilhar informações.

## O que é LAN?

LAN (Local Area Network) é uma rede local, usada em casas, escolas e empresas.

## O que é WAN?

WAN (Wide Area Network) conecta redes em grandes distâncias, como a Internet.

## O que aprendi hoje?

- O que é uma rede
- Diferença entre LAN e WAN
- Como dispositivos se comunicam
