# Dia 2 - Princípios de Comunicação

Data: 19/06/2026

## Objetivo do módulo

Explicar a importância dos padrões e protocolos nas comunicações de rede.

---

## O que são protocolos de comunicação?

Protocolos são conjuntos de regras que permitem que dispositivos se comuniquem corretamente em uma rede.

### Exemplos

- HTTP
- HTTPS
- DNS
- DHCP
- TCP
- UDP

---

## O que são padrões de comunicação?

Padrões garantem que dispositivos de diferentes fabricantes consigam se comunicar.

### Exemplos

- IEEE
- Ethernet
- Wi-Fi

---

## Modelos de comunicação de rede

### Modelo OSI

Possui 7 camadas:

1. Física
2. Enlace de Dados
3. Rede
4. Transporte
5. Sessão
6. Apresentação
7. Aplicação

---

### Modelo TCP/IP

Possui 4 camadas:

1. Acesso à Rede
2. Internet
3. Transporte
4. Aplicação

---

## O que aprendi hoje?

- O que são protocolos de comunicação
- O que são padrões de comunicação
- Diferença entre modelo OSI e TCP/IP
- Importância dos protocolos em redes
